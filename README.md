# Query Plan Analyzer and Visualization Tool
 * This tool analyze the PostgreSql queries using EXPLAIN statement.
 * and then textual output(json) will transformed into tree based diagram using d3.js and EXPLAIN(analyze,format json) query.

### Steps to Run the tool:
 
 - first run the server using command: $npm run dev
 - then go live using index.html file on Visual Studio Code IDE.
